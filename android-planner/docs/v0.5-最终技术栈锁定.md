│  └──────────────────────┬────────────────────────────────┘  │
│                         │ 加密                              │
│  ┌──────────────────────▼────────────────────────────────┐  │
│  │  SQLCipher (全库加密)                                  │  │

## 三、密钥派生流程

```
用户输入主密码
       ↓
生成 salt (16 bytes, 存储在 Keystore)
       ↓
Argon2id(password, salt, 64MB, 3, 1)
       ↓
输出 256-bit master_key
       ↓
┌──────────────────────────────────────────┐
│  master_key 用于:                         │
│                                          │
│  1. SQLCipher passphrase (全库加密)       │
│     → Keystore 加密后存储在 DataStore     │
│                                          │
│  2. 字段级加密密钥 (AES-256-GCM)          │
│     → 派生: HKDF(master_key, "field")    │
│                                          │
│  3. 导出文件加密密钥                       │
│     → 派生: HKDF(master_key, "export")   │
└──────────────────────────────────────────┘
```

## 四、数据库安全设计

### SQLCipher 配置

```kotlin
// 全库加密配置
val passphrase = KeystoreManager.decrypt(
    DataStoreManager.getEncryptedDbPassphrase()
)
val db = Room.databaseBuilder(context, AppDatabase::class.java, "toolbox.db")
    .openHelperFactory(SupportFactory(passphrase))
    .build()
```

### 字段级加密（密码本额外层）

```kotlin
@Entity
data class PasswordEntry(
    @PrimaryKey val id: String,
    val title: String,                    // 明文（非敏感）
    val groupId: String,                  // 明文
    val icon: String?,                    // 明文

    // Level 0: 密码原文 — 字段级 AES-256-GCM
    @ColumnInfo(name = "password_encrypted")
    val passwordEncrypted: ByteArray,     // 密文
    @ColumnInfo(name = "password_iv")
    val passwordIv: ByteArray,            // 初始向量
    @ColumnInfo(name = "password_tag")
    val passwordTag: ByteArray,           // GCM 认证标签

    // Level 1: 提示 — 字段级 AES-256-GCM
    @ColumnInfo(name = "hints_encrypted")
    val hintsEncrypted: ByteArray,
    @ColumnInfo(name = "hints_iv")
    val hintsIv: ByteArray,
    @ColumnInfo(name = "hints_tag")
    val hintsTag: ByteArray,

    // Level 2: 扩展 — 字段级 AES-256-GCM（预留）
    @ColumnInfo(name = "ext_encrypted")
    val extEncrypted: ByteArray?,
    @ColumnInfo(name = "ext_iv")
    val extIv: ByteArray?,
    @ColumnInfo(name = "ext_tag")
    val extTag: ByteArray?,

    val createdAt: Long,
    val updatedAt: Long,
    val deletedAt: Long?
)
```

### 加密层次总结

```
Layer 1: Android Keystore (硬件)
  └── 保护 master_key 和 DB passphrase

Layer 2: SQLCipher (软件)
  └── AES-256 加密整个数据库文件

Layer 3: 字段级加密 (应用层)
  └── AES-256-GCM 加密密码本敏感字段

三层保护，任一层被绕过仍有上层防线
```

## 五、提醒系统兼容性策略

```
Android 版本兼容性矩阵:

API 26-30 (Android 8-11):
  ✅ SCHEDULE_EXACT_ALARM 默认授予
  ✅ 后台服务限制但可用

API 31 (Android 12):
  ⚠️ 需声明 SCHEDULE_EXACT_ALARM 权限
  ⚠️ 用户可在设置中撤销
  → 降级: 退化为 WorkManager 不精确调度

API 33 (Android 13):
  ⚠️ 需运行时申请 POST_NOTIFICATIONS
  → 启动时引导用户授权

API 34 (Android 14):
  ⚠️ 前台服务需声明类型 (FOREGROUND_SERVICE_SPECIAL_USE)
  → 在 manifest 中声明 specialUse + 用户引导

API 35 (Android 15):
  ⚠️ 闹钟权限进一步收紧
  → 使用 canScheduleExactAlarms() 检测，不可用时降级

降级策略:
  精确闹钟可用 → AlarmManager (最佳)
  精确闹钟不可用 → WorkManager + 15分钟最小间隔
  通知被禁 → 前台服务 + 振动 (兜底)
```
